Soluciones de WaterSense

Herramientas digitales para reportar, monitorear y analizar el acceso al agua segura.

Reporte de problemas

Permite registrar fallas como cortes de agua, baja presión, agua turbia o mal olor.

Mapa hídrico

Visualiza zonas críticas según el nivel de gravedad, ubicación y reportes enviados por la comunidad.

Guias

Recomendaciones sencillas para purificar, almacenar y consumir agua de forma segura.

Datos Historicos

Organiza reportes y recopila datos historicos para que investigadores y autoridades puedan analizar la situación hídrica.
